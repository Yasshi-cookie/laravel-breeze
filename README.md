# このリポジトリの作成方法
laravel/breezeを使います。
まず、laravelプロジェクトを作成
```bash
$ composer create-project laravel/laravel your-project-name
```

laravel/breezeをインストール
```bash
$ composer require laravel/breeze --dev
```

```bash
$ php artisan breeze:install
```


参考：https://laravel.com/docs/10.x/starter-kits
